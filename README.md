# Fungio Ergo Cogito Ergo Sum

A comprehensive exploration of fungal consciousness and its role in evolution - the hidden partnerships that shape all life.

## Deployment

This site is automatically deployed to Netlify when changes are pushed to the main branch.

**Live Site:** https://fungioergosum.com

## Local Development

Simply open `index.html` in your browser to view locally.

## Files

- `index.html` - Main manifesto content
- `fungio_style.css` - Styling for the site
